    HTML #

    HTML — это не продолжатель языка разметки гипертекста, а новая открытая платформа, предназначенная для создания веб-приложений использующих аудио, видео, графику, анимацию и многое другое.

    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> <html> <head> <meta http-equiv="Content-Type" content="text/html; charset=utf-8"> <title>Моя первая веб-страница</title> </head> <body> <h1>Заголовок страницы</h1> <p>Основной текст.</p> </body> </html>

    Что нужно чтобы начать?

    1. В Windows откройте программу Блокнот
    2. Наберите или скопируйте код в Блокноте
    3. Сохраните готовый документ с расширением html
    4. Откройте сохранённый файл в любов Браузере

    Поздравляю! Вы написали свой первый сайт!

    Источник: Htmlbook.ru

    CSS #

    Давайте стилизуем наш файл?

    CSS — это язык, описывающий стиль HTML-документа. CSS описывает, как должны отображаться HTML-элементы.

    В нашем файле, внутри тега <head></head> напишем тег <style></style>. И уже внутрь него давайте поместим следующий код.

    body { background-color: lightblue; } h1 { color: white; text-align: center; } p { font-family: verdana; font-size: 20px; }

    Неплохо получилось!

    Источник: Html5css.ru

    JS #

    Давайте посмотрим, что такого особенного в JavaScript, чего можно достичь с его помощью и какие другие технологии хорошо с ним работают.

    Что такое JavaScript?

    Изначально JavaScript был создан, чтобы «сделать веб-страницы живыми». Программы на этом языке называются скриптами. Они могут встраиваться в HTML и выполняться автоматически при загрузке веб-страницы. Скрипты распространяются и выполняются, как простой текст. Им не нужна специальная подготовка или компиляция для запуска.

    К практике!

    В тоь же файле, только внутри тега <body></body> напишем тег <script></script>. А в нём такой код.

    <script> alert('Привет мир!'); </script>

    Отлично!

    Источник: Learn.javascript.ru
